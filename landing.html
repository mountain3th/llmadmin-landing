常见问题

什么是 LLM Admin？ expand_more
LLM Admin 是一款专为开发者和企业设计的大模型管理与分发平台，攨合了多种协议与供应商，提供极致的性能和灵活性。
怎么自定义路由？ expand_more
您可以在控制台通过直观的界面或配畮文件定义流量路径，支持基于模型类型、地理位置或优先级的智能调度。
统一接口是什么？ expand_more
统一接口指将不同供应商（如 OpenAI, Claude, 国产模型等）的 API 标准化，让开发者只需一套代码即可调用所有模型。
具体路由的规则是怎样的？ expand_more
路由规则采用多层级匹配机制，包括负载均衡、故障自动转移以及基于 Token 消耗的成本优化规则。
